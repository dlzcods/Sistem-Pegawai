# Sistem Pegawai
Project ini dibuat untuk memenuhi tugas dari mata pelajaran **Pemrograman Berorientasi Objek.**
Project ini berisi form sebagai berikut:
* Login
* Menu
* Divisi
* Jabatan
* Pegawai
* Gaji Pegawai
## Fitur
Fitur di form *Divisi, Jabatan, Pegawai, Gaji Pegawai.*
| Name  | Status |
| ----- | --- |
| Tambah | ✅ |
| Update | ✅ |
| Hapus | ✅ |
| Search| ✅ |
